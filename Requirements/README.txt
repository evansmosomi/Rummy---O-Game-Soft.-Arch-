In this folder you can find 3 files aside from this one which contain no code but serve the purpose of gathering the requirments to design the game.

The first file is a Visal Paradigm file where all the requirments where listed and have the uses cases that inspired each requirment in their description.

The second file is the Requirments review report, part of the granding criteria. The purpose it serves is to thouroughly explain each requirment and to update them if they need improvment.

The third and last file is a list of all the requirments in table form, also part of gradeing criteria. The purpose for this file is to have easy access to all the requirments in a compact
and easy to access location
